# Extreme Strategy Web Analysis

익스트림 제품의 재구매율 분석과 최적 샘플링 동봉 전략을 시각화하는 웹 애플리케이션입니다.

## 주요 기능
- **재구매율 순위 요약**: 60일/90일 기준 브랜드 내 제품별 재구매율을 실시간으로 확인 및 정렬할 수 있습니다.
- **최적 샘플링 동봉 전략 분석**: 특정 제품을 샘플로 증정할 때, 어떤 제품을 구매한 고객에게 제공해야 실구매 전환율이 가장 높을지 분석 가이드를 제공합니다.
- **앵커 제품 추천**: 앵커 제품 선택 시 데이터 분석을 통한 궁합이 좋은 추천 제품과 감사카드 문구를 생성합니다.

## 배포 방법 (Vercel)
1. GitHub 리포지토리를 생성하고 이 프로젝트 소스를 푸시합니다.
2. [Vercel](https://vercel.com)에 로그인 후 "Add New > Project"를 선택합니다.
3. 해당 GitHub 리포지토리를 연결(Import)합니다.
4. "Deploy" 버튼을 클릭하면 수초 내에 웹 URL이 생성됩니다.

## 기술 스택
- HTML5 / Vanilla CSS / JavaScript
- 데이터 시각화 및 동적 렌더링
- Vercel 호스팅 아키텍처
